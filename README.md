# HerbVerse Vendor App

This mobile app is designed for vendors to manage their herbal product listings and sales on the HerbVerse platform, created for the HackVerse Hackathon.

---

## Project Overview

HerbVerse Vendor empowers herbal product sellers to easily upload products, manage orders, and connect directly with customers through a dedicated interface. The app streamlines vendor operations in the digital herbal marketplace.

---

## Key Features

- Vendor registration and secure login
- Add, update, and delete herbal product listings
- Manage incoming orders and sales details
- Receive notifications for new orders
- Built with React Native for a smooth mobile experience

---

## Technologies

- React Native for mobile development
- Firebase for backend services and authentication
- JavaScript programming
